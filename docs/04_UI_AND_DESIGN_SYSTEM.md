# UI/UX Design System

## 1. Design Philosophy

The entire interface must feel like a premium automotive brand rather than a generic SaaS product.

The design language should combine the simplicity of Apple, the elegance of Porsche, the precision of Mercedes-Benz and the modern digital experience of Linear.

Users should immediately feel that they are browsing a high-end vehicle showroom.

The interface must communicate:

- Premium
- Elegant
- Minimal
- Confident
- Timeless
- Carefully designed

Avoid anything that feels template-based.

---

## 2. Visual Identity

The interface should be extremely clean.

Large whitespace is encouraged.

Every element must have a clear purpose.

The design should prioritize visual hierarchy instead of decorative elements.

Avoid unnecessary gradients, bright colors or excessive effects.

Less is more.

---

## 3. Layout Principles

Never generate a generic SaaS landing page.

Avoid layouts that simply stack:

Title

Subtitle

Button

Instead:

- Use editorial composition
- Large premium vehicle photography
- Asymmetrical layouts where appropriate
- Sections with different compositions
- Strong visual hierarchy
- Wide spacing between sections

Each section should feel intentionally designed.

Avoid repetitive card grids.

---

## 4. Hero Section

The first screen must occupy the full viewport.

Use a cinematic luxury vehicle image as the main visual element.

Navigation should float above the hero with a transparent or glass background.

Hero typography should be bold, large and minimal.

One primary CTA is enough.

The hero should immediately communicate luxury and trust.

---

#FFFFFF

#111111

#5F6368

Photography should be the strongest visual element.

Primary Font

Geist

Monospace

Geist Mono

Hero Titles

56–72px

Section Titles

36–48px

Body

16–18px

Small Labels

13–14px

Typography should feel editorial rather than corporate.

---

## 7. Components

Cards

- Large rounded corners (20–28px)
- Very soft shadows
- Thin borders
- Premium spacing
- Large images
- Minimal text

Buttons

- Rounded
- Medium height
- Subtle hover
- No flashy gradients

Inputs

- Clean
- Spacious
- High contrast
- Large touch targets

Navigation

- Floating
- Minimal
- Transparent on hero
- Solid after scrolling

---

## 8. Motion Design

Animations should feel invisible.

Duration

150–250ms

Easing

ease-in-out
or
cubic-bezier(.22,.61,.36,1)

Hover

- Slight elevation
- Image zoom (1.02)
- Border transition
- Soft shadow

Never use dramatic animations.

Avoid bounce effects.

Avoid excessive motion.

---

## 9. Loading States

Never use traditional spinners.

Always use Skeleton Loading.

Skeletons must match the final layout.

Loading should feel smooth and premium.

---

## 10. Feedback

Toast notifications should slide gently from the corner.

No modal interruptions unless absolutely necessary.

Success

Elegant

Error

Clear but calm

Warning

Subtle

---

## 11. Photography

Vehicle photography is the centerpiece of the design.

Use:

- Studio lighting
- Cinematic angles
- High resolution
- Premium composition
- Natural reflections

Avoid low-quality stock images.

Avoid inconsistent image styles.

---

## 12. What To Avoid

Never generate Bootstrap-looking pages.

Never create generic SaaS templates.

Never center every section.

Never repeat identical layouts.

Never overuse gradients.

Never use loud colors.

Never overcrowd the interface.

Never rely on icons instead of typography.

Never sacrifice whitespace.

Avoid dashboard aesthetics for public-facing pages.

---

## 13. Inspiration

The visual language should feel inspired by:

- Apple
- Porsche
- Mercedes-Benz
- Polestar
- Lucid Motors
- Rivian
- Nothing
- Linear
- Stripe
- Vercel

Do not copy these brands.

Only capture their attention to spacing, hierarchy, typography and premium feeling.

---

## 14. Final Rule

Every screen should look like it was crafted by an experienced product designer, not generated from a UI template.

If any screen resembles a standard SaaS dashboard, redesign it until it feels custom, premium and automotive.
## Design Quality Rules

Before implementing any page:

- First propose 3 different visual concepts.
- Choose the strongest concept.
- Explain why it fits the premium automotive brand.
- Only then begin implementation.

Every new page must introduce at least one unique layout idea.

Do not reuse the same hero, card, section or grid across pages unless intentionally part of the design system.

Think like a Senior Product Designer, not a code generator.
